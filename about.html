Emoji Spelling Bee™


Welcome to the Emoji Spelling Bee™! This is a fun and interactive game where you can test your spelling skills using emojis. The game presents you with a series of emojis, and your task is to spell out the word that they represent. It's a great way to learn new words and have fun at the same time!


It combines the joy of spelling games and the fun of emojis into a game that is way more fun then math.


The 'Emoji Spelling Bee™' is a project by S.H.

Roadmap


Currently Emoji Spelling Bee is only available as a download in .py format. It will only work correctly if you have the correct libraries installed which is a little cumbersome for the average user to install and then utilize.

We would like to see this instead deployed as web-based game.

This would require a little bit more development and refinement for a back-end python and some sort of front-end html. This will require work.
